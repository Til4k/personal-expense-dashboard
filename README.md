# 📊 Personal Expense Analytics Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard built to analyze personal spending behavior and financial trends using a real-world expense dataset.

The dashboard provides insights into:
- Total spending
- Average expenses
- Monthly spending trends
- Spending by category
- Payment method usage

---

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📈 Dashboard Features
- KPI Cards for Total Spending & Average Expense
- Category-wise Expense Breakdown
- Monthly Spending Trend Analysis
- Payment Method Usage Analysis
- Interactive Month Slicer
- Clean and responsive dashboard layout

---

## 🧹 Data Cleaning & Transformation
During the project, several preprocessing steps were performed:
- Date format correction using Locale settings
- Handling inconsistent date formats
- Creating DAX calculated columns
- Fixing month sorting issues using MonthNumber column

---

## 📊 Key Insights
- Food and Shopping are among the highest expense categories
- Spending peaks during mid-year months
- Cash is the most frequently used payment method
- Monthly spending declines toward the end of the year
